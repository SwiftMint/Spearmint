### WoW Terminal

Anyone can access both the most updated API **and** Textures by creating a shortcut to the `WoW.exe` application. In the file-path, append ` -console`.

Use that shortcut to open World of Warcraft and the title screen will appear. Press the tilde (`~`) on the keyboard and a terminal will open up.

Type and enter one (or both) of the following commands. The terminal will offer auto-complete!

|Command|Files|Time|
|-|-|-|
|**ExportBlizzardInterface**|BlizzardInterfaceCode|A few seconds.|
|**ExportBlizzardArt**|BlizzardInterfaceArt|A minute, depending on processing speed.|

The files will appear in the **`World of Warcraft`** > *`_retail_`* folder.
