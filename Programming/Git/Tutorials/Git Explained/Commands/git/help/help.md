# `--help` | [Command Description](https://git.github.io/htmldocs/git-help.html)
##  `-h` <br> `--help` <br> `help`
### Describe the associated command.

**Exception: `git`**
Since `git` is the parent of `-h`, it must go first.
`git -h` results in the same output as `git`, as seen in `git.md`.

Each subsequent command/subcommand will utilize `-h` when necessary.

See: `all.md`
